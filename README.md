## About ABAP

ABAP (Advanced Business Application Programming) is SAP's proprietary programming language, used to build and customize backend logic for SAP business systems. It's the language behind a huge portion of the world's enterprise software, which makes it a valuable (if niche) skill for backend development.

## Why This Repo

- Build a portfolio that demonstrates hands-on ABAP experience
- Keep a personal reference of solved problems and patterns I've learned
